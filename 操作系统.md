# 1. 命令行

## 1.1 四要素：

1. 可执行程序
2. 参数
3. 工作路径
4. 环境变量

## 1.2 shell:

本质是一个进程，响应用户输入。

- 大多数引用程序执行时会fork一个子进程，继承父进程的环境变量
  - `java` `maven` `python`  `sh`  `./`
  - 这个命令修改了环境变量只会在他们的子进程中有效，不会影响父进程

- `source` `exec`会影响当前的环境变量

## 1.3 管道

任何命令都绑定了三个输入输出

- 标准输入：fd = 0
- 标准输出: fd = 1
- 错误输出: fd = 2
